# depth-estimation
Depth Estimation from a Monovision Camera using a Deep Neural Network

### Utilities
- deep_utils.py
- image_depth.py
- pickle_dataset.py
- rosbag_utils.py

### Dataset
- [FieldSAFE](https://vision.eng.au.dk/fieldsafe/)

- [NYU Depth Dataset V2](https://cs.nyu.edu/~silberman/datasets/nyu_depth_v2.html)
